# Logo Assets

This folder contains the main Monytix logo files.

## Current Files

- `transparency-01.png` - Main logo with transparency (upload here)

## Usage with Path Alias

```javascript
// Import using alias
import MonytixLogo from '@/logo/transparency-01.png';

// Use in component
function Header() {
  return <img src={MonytixLogo} alt="Monytix - Make Every Paisa Count" />;
}
```

## Logo Specifications

- **Brand**: Monytix
- **Tagline**: "MAKE EVERY PAISA COUNT"
- **Colors**: Golden gradient text with colorful "O" (blue, yellow, orange)
- **Background**: Transparent
- **Format**: PNG with transparency support
